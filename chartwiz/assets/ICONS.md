# App Icons

Place your ChartWiz icon files here before running `npm run build:app`.

Required files:
- `icon.ico`  — Windows (256×256 recommended, multi-size ICO)
- `icon.icns` — macOS  (512×512 recommended)
- `icon.png`  — Linux  (512×512 PNG)

To generate these from your SVG logo:
1. Export a 1024×1024 PNG from your design tool
2. Use https://www.icoconverter.com/ for .ico
3. Use https://cloudconvert.com/png-to-icns for .icns
4. The .png is used directly for Linux
